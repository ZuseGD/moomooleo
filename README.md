### **Project Overview**

This repository contains a statistical analysis pipeline designed to interpret game meta-data and optimal build strategies for the Champions Meeting 4 event. The project utilizes machine learning libraries to identify performance correlations and predictive factors for competitive gameplay outcomes.

**ðŸ”— Project Repository:** [GitHub Link](https://github.com/ZuseGD/cm4-analysis)

-----

### **Key Achievements**

  * **Statistical Pipeline:** Designed a robust analysis workflow using **Python** to process raw game logs.
  * **Meta-Analysis:** Conducted deep-dive analysis on game mechanics to identify win-rate factors and optimal character builds.
  * **Predictive Modeling:** Extracted key features from raw datasets to model potential gameplay outcomes.
